input-password
directive in module input-password

Description

Password component. Features:

  • Validation rules. You can specify rules to validate password
  • Compliant with form validation. if inserted in a form, it will manage the validation of the form
  • Strength. You can specify a function to compute strength

Usage

as element:
<input-password
       ng-model="{string}"
       name="{string}"
       rules="{array}"
       strength="{function}">
</input-password>

Parameters

ParamTypeDetails
ngModelstring

the password.

namestring

name of the field (see form validation).

rulesarray

array of rules.

  • id(optional): identifier of the rules (see form validation)
  • caption: displayed message
  • validator: function or RegExp. will return true if the rule is respected
strengthfunction

function(value) to define the strength of the password. it will return a float [0;1]

(default: null)
